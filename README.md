Vue (pronounced /vjuË/, like view) is a progressive framework for building user interfaces.
This project with vue 3 of vue.js + firebase
* GitHub : https://github.com/edenko/vue-project


## Project setup
```
npm install -g @vue/cli
npm install -g @vue/cli-init
vue create [name of project]
```


## Run
```
npm run serve
```
Load `http://localhost:8080` to test the endpoint.


## Requirements
* node (>= 10.5.0)
* vue 3
